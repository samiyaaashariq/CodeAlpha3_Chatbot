# CodeAlpha3_Chatbot
Smart Chatbot

A simple Python chatbot that interacts with users through text-based conversations.

Features

* Greets the user
* Answers basic questions
* Tells random jokes
* Shows current date and time
* Handles unknown inputs
* Allows users to exit the chat

Sample Output

You: hello

Bot: Hello! Nice to meet you.

You: what is your name

Bot: I am a Python Chatbot created for the CodeAlpha Internship.

You: joke

Bot: Why do programmers prefer dark mode? Because light attracts bugs!

You: time

Bot: Current time is 07:15 PM

You: bye

Bot: Goodbye! Have a wonderful day.

Technologies Used

* Python

Run the Project

py chatbot.py
